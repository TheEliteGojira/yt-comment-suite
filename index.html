01 — API Key

Get a free key at Google Cloud Console → Enable "YouTube Data API v3" → Credentials → Create API Key. Your key is saved locally in your browser so you don't have to re-enter it. Free quota: ~10,000 units/day (each page of 100 comments = 1 unit).
